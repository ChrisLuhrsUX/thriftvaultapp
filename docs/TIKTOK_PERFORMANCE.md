# TikTok Performance Log

Companion to [TIKTOK_STRATEGY.md](TIKTOK_STRATEGY.md). Append one entry per post; update view counts at the 1hr / 6hr / 24hr / 7-day marks so we can see the velocity curve, not just the final number.

**How to update:** tell Claude "log tiktok post" with whatever data you have. Claude appends a new entry or updates the latest one.

**Verdict bands** (rough, for the @thriftvaultapp account at current follower count):
- **Hit:** >2k views at 24hr OR >5% like rate OR >1% comment rate
- **Solid:** 500–2k views at 24hr with >3% like rate
- **Flat:** 200–500 views, low engagement
- **Miss:** <200 views OR negative comments

Numbers calibrate over time as the account grows.
